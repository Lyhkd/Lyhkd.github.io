Structured Prediction

分类/回归问题：输入是任何对象，输出是离散或实数

结构化预测：输入是任何对象，输出是复杂结构对象

Markov Random Field

Potential A potential φ(x) is a non-negative function of the variable x. A joint potential φ(x1, x2, . . . ) is a non-negative function of a set of variables.

Markov Random Field (MRF) = Markov Network For a set of variables X = {x1,…,xM} a Markov Random Field is defined as a product of potentials over the (maximal) cliques {Xk}Kk=1 of the undirected graph G

  • The factorization is not unique: an MRF can have multiple different factorizations

  • If all potentials are strictly positive: Gibbs distribution
