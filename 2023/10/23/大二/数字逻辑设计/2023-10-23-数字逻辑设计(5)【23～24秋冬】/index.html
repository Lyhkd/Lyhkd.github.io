Beginning Hierachical Design

把三位的模块在九位设计里面调用称为“实例化”——对于一个给定的可以重复使用的模块，这个模块被称为一个实例（instance），他的应用称为实例化（instantiation）

    1. 使电路变得简单
    2. 预定义的原子模块用符号而不是用逻辑图表示
    3. 模块可以重复使用
  • Reusable Functions：Whenever possible, we try to decompose a complex design into common, reusable function blocks
    These blocks are verified and well-documented, placed in libraries for future use
  • Top-Down versus Bottom-Up：两种设计方法，一般都是采用top-down的方式

3.6.3 Enabling Function

多出一个EN信号的输入，

  • Enabling permits an input signal to pass throughto an output
  • Disabling blocks an input signal from passing through to an output, replacing it with a fixed value(Hi-Z, 0, 1)

The value on the output when it is disable can be Hi-Z(three-state buffer, transportation gate)

连接EN的AND门，当EN为0时，输出恒为0

连接"EN非"的OR门，当EN输入为0时，输出恒为1

3.7 Decoding

n-bit input code 到 m-bit output code

functional blocks for decoding are called n-to-m line decoders, where m≤2nm\leq 2^nm≤2n

Decoders Expansion

要想搭出一个n输入的decoder，只需要：

如果n为偶数，则将两个n/2的decoder搭在一起

如果n为奇数，则将一个n+12\frac{n+1}{2}2n+1​decoder 和一个n−12\frac{n-1}{2}2n−1​decoder搭在一起

如此递归，直到n为1

计算decoder需要的inverters和and：inverters数等于n，and门数一层层叠加计算（因为默认AND门二输入）

Decoder Enable

为什么加一个EN信号，需要额外增加一层？

分配电路：将来自一条输入线上的信息传送到2^n条输出线中的任意一条。执行这种分配任务的电路叫做多路分配器

Decoder and OR gate

通过Decoder和OR gate可以构成任何一个逻辑函数

两个方法

  1. 找到最小项，把对应1的项或起来
  2. 找到对应0的项，使用或非门
  • Decoder不一定是一个好的选择方案，在一些情况下会造成很大的浪费。

3.8 Encoding

以上定义的编码器有一个限制条件，那就是任何时候只允许一个输人是活动的:如果同时有两个输入是活动的，输出将产生错误的结果。

Priority Encoder

当两个或多个输入同时等于1时，优先级最高的输入优先被处理。

3.9 Selecting using Multiplexers

多路复用器是一个组合电路，它可以从多个输入中选 一个输入，并将信息直接传输到输出。选择哪一条输入线由一组输入变量控制，它们被称为选择输入 ( sc lecti on input )

一个n-2^n译码器、 一个使能电路(n * 2 AND gate)和一个或门组成

多重多路复用器

用多路复用器来实现各个函数：

  1. 找出真值表
  2. 把各个输入接到MUX的selector
  3. 把input接对应的输出值

可以用一个nto1的MUX 或者 一个n-1to1的MUX + 一个inverter 来实现任何n输入的布尔函数
