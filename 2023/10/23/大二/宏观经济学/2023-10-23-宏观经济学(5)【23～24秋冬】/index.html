三、两部门经济:有效需求的原理和框架

(一)有效需求的原理

  1. 在总供求均衡决定均衡国民收入原理基础上，强调短期内的供给过剩和需求不足。

  2. 强调总需求(有效需求)的决定性作用。

  3. 总需求(有效需求)由四个部分组成: 消费需求、投资需求、国外需求、政府需求

(二)基本假设

  1. 经济中只有居民和企业两个部门(I投资外生给定，作为常量);

  2. 社会总能够以不变的价格提供适应需求的产品与服务;

    “凯恩斯定律”: “有效需求总能得到相应的供给”，而且社会需求的变动只会引起产量变动，不会引起价格变动。
    库存量在一开始消耗并不大，企业不会因为需求的增加而短时间内提高价格

  3. 利率保持不变；

  4. 社会上没有折旧和公司未分配利润;

  5. 只考虑短期;

(三)凯恩斯主义交叉图

Y=C+IY=C+I Y=C+I

C为消费，Y为收入，I为利率

(四)投资等于储蓄

E=C+IY=C+SC+I=Y=C+SI=SE = C+I\\Y=C+S\\C+I=Y=C+S\\I=S E=C+IY=C+SC+I=Y=C+SI=S

  1. 两部门经济

    只要居民收入从国民经济大循环中出来（储蓄等)，那么经济就会衰退。因为居民的消费在不断衰减，企业能够获得的利润也在减少。
    ==为什么生产出了1000亿价值的东西，以900亿卖掉了？==

    • 经济衰退时希望股票市场活跃：1. 股票能起到为企业融资的作用 2. 股票市场繁荣能够促进经济消费

    • 以创新创业把储蓄转化成投资泵回经济循环。

      1. 总需求<总供给：降低利率
      2. 总需求>总供给：加息

  2. 三部门经济

    • 若需求<供给：可采用1. 降低利率，但也不能太低，因为会有流动性陷阱 2. 调整政府收支，保持财政赤字，多支出

      赤字，意即入不敷出。财政赤字，即政府的支出额度比收入的部分多，多花的这一部分预算，就会随着国库的支出流入国内社会当中；贸易赤字，即进口总额比出口总额多，又称入超，多花的这一部分钱，就随着国际贸易流入国际社会当中。

    • 若需求>供给：1. 加息 2. 政府压缩消费

  3. 四部门经济

    M：进口 X：出口（可以写成NX）
    T-G 政府调节因子
    M-X 贸易调节因子

    • 总需求<总供给：1. 降低利率 2. 政府财政赤字 3. 增大贸易顺差（X增大）（很难的！！会导致其他国家的贸易逆差。）
    • 总需求>总供给：1. 提高利率 2. 加税 3. 对贸易逆差视若无睹

二、投资乘数和乘数原理

写出需求函数

Y=C+IC=α+βYY=α+I1−βkI=11−βY=C+I\\ C=\alpha + \beta Y\\ Y = \frac{\alpha + I}{1-\beta}\\ k_I = \frac1{1-\beta} Y=C+IC=α+βYY=1−βα+I​kI​=1−β1​

因为0<β<10<\beta<10<β<1，乘数kIk_IkI​大于1
一般情况下，投资乘数都是等于消费乘数的。

这里消费者支出乘数的外生变量是α(不受收入影响的固定消费)，内生变量指y（收入）

三、其他乘数

涉及三部门和四部门经济中的乘数

Y=C+I+GC=α+βYd=α+β(Y−T)Y=α+I+G−βT1−βkc=kI=11−βY = C+I+G\\ C = \alpha + \beta Y_d = \alpha + \beta(Y-T)\\ Y=\frac{\alpha+I+G-\beta T}{1-\beta}\\ k_c = k_I=\frac1{1-\beta} Y=C+I+GC=α+βYd​=α+β(Y−T)Y=1−βα+I+G−βT​kc​=kI​=1−β1​

  1. 政府购买乘数

    kG=11−βk_G=\frac1{1-\beta} kG​=1−β1​

    政府购买里面的各项支出都是类似个体消费（购买苹果、打印纸）和民间投资（修公路）的

  2. 税收乘数

    kT=−β1−βk_T=\frac{-\beta}{1-\beta} kT​=1−β−β​

    在经济衰退的时候，不能加税！

    都是常数项发生变动时，引起收入变化。

    ==这到底是税收乘数还是转移支付乘数啊啊==

    Y=C+I+GC=α+βYd=α+β(Y−T−tY+Ttr)Y=α+I+G−βT+βTtr1−β(1−t)kT=−β1−β(1−t)Y = C+I+G\\ C = \alpha + \beta Y_d = \alpha + \beta(Y-T-tY+T_{tr})\\ Y=\frac{\alpha+I+G-\beta T+\beta T_{tr}}{1-\beta(1-t)}\\ k_{T} = \frac{-\beta}{1-\beta(1-t)} Y=C+I+GC=α+βYd​=α+β(Y−T−tY+Ttr​)Y=1−β(1−t)α+I+G−βT+βTtr​​kT​=1−β(1−t)−β​

    实际上有一部分是比例税（与固定税相对）

    考试中，固定税常设为0

  3. 政府转移支付乘数

    Y=C+I+GC=α+βYd=α+β(Y−T+Ttr)Y=α+I+G−βT+βTtr1−βktr=β1−βY = C+I+G\\ C = \alpha + \beta Y_d = \alpha + \beta(Y-T+T_{tr})\\ Y=\frac{\alpha+I+G-\beta T+\beta T_{tr}}{1-\beta}\\ k_{tr} = \frac{\beta}{1-\beta} Y=C+I+GC=α+βYd​=α+β(Y−T+Ttr​)Y=1−βα+I+G−βT+βTtr​​ktr​=1−ββ​

    比正常消费乘数要小。==原因在书上==

    转移支付是政府无偿直接给你，性质和税收相同，所以互为相反数

    转移支付是指政府或企业的一种不以购买本年的商品和劳务而作的支付，即政府或企业无偿地支付给个人或下级政府，以增加其收入和购买力的费用。 它是一种收入再分配的形式。 转移支付是包括养老金、失业救济金、退伍军人补助金、农产品价格补贴、公债利息等政府与企业支出的一笔款项。

  4. 平衡预算乘

    ΔY=kGΔG+kTΔT=\Delta Y = k_G\Delta G+ k_T\Delta T = ΔY=kG​ΔG+kT​ΔT=

    加入比例税后，是否还为一？不考虑转支付

  5. 对外贸易乘数

    Y=C+I+G+NXC=α+βYd=α+β(Y−T−tY+Ttr)Y=C+I+G+NX\\ C=\alpha +\beta Y_d = \alpha+\beta(Y-T-tY+T_{tr})\\ Y=C+I+G+NXC=α+βYd​=α+β(Y−T−tY+Ttr​)

    (T表示固定税，t是比例税税率。由于自变量增量变化引起因变量Y增量变

    化，总是用其与Y无关部分的变化，如用α表示C。为了简化，可以直接把固定税写成T，税收乘数就是ΔYΔT\frac{\Delta Y}{\Delta T}ΔTΔY​。后面的对外贸易乘数也是同理。)

    乘数变小了！居民会用一部分收入购买进口产品，导致国内消费减少，也就导致国内的投资、生产减少，也就收入减少。
